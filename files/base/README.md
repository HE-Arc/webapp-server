## MySQL

Connect to MySQL (hint: `echo \$PASSWORD`):

```
$ mysql --host $MYSQL_HOST --user $GROUPNAME --password
```

## Postgres

Connect to Postgres:

```
$ psql -h $POSTGRES_HOST -U $GROUPNAME
```
